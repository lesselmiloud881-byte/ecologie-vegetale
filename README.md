L'écologie végétale est une branche de l'écologie qui étudie les interactions entre les plantes et leur environnement, ainsi que les relations entre les différentes espèces végétales. Elle cherche à comprendre comment les plantes interagissent entre elles, avec les autres organismes vivants (comme les animaux, les champignons et les micro-organismes) et avec les facteurs abiotiques (comme la lumière, la température, l'eau, les sols)
